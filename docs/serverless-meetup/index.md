Serverless 场景目前也是大厂在推，主要有以下几个场景：

- 异步并发，组件可独立部署和扩展的场景。
- 突发或服务使用量不可预测的场景。
- 短暂、无状态的应用，对冷启动时间不敏感的场景。
- 需要快速开发、迭代的业务。
